# Spec — Dashboard

## Goal

Provide the owner with a simple admin interface.

## First screen

Calendar view.

## Main modules

- Calendar
- Stay requests
- Reservations
- Pricing
- Property content
- Photos
- Activity

## Calendar states

- Available
- Pending request
- Reserved
- Blocked

## Definition of Done

- Owner can approve/refuse requests.
- Approved request creates reservation.
- Reservation blocks dates.
- Owner can block/unblock dates.
- Owner can edit pricing.
