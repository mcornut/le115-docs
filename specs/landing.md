# Spec — Landing

## Goal

Build the public one-page website for `Le 115, Maison de Provence`.

## Required sections

1. Header
2. Hero
3. Key facts
4. Description
5. Amenities
6. Gallery
7. FAQ
8. Location
9. Stay planner
10. Footer

## Main CTA

`Estimer mon séjour`

The CTA scrolls to the stay planner section.

## Design constraint

Stay visually close to the provided communication mockup.

## Components

- Header
- HeroSection
- KeyFactsSection
- DescriptionSection
- AmenitiesSection
- GallerySection
- FAQSection
- LocationSection
- StayPlannerSection
- Footer

## Definition of Done

- Responsive desktop/mobile.
- FR/EN content supported.
- CTA scrolls correctly.
- Gallery opens fullscreen.
- Stay planner section is reachable from header CTA.
